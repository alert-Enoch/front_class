태그 두가지 종류
    1. 내용(자식)을 가질 수 있는 태그
        여는태그 닫는태그로 구성되어있고, 그 사이에 자식을 넣어준다
        <h1>안녕하세요</h1>
        <div>
            <h1>안녕하세요</h1>
        </div>
    2. 자식을 가질 수 없는 태그
        열리면서 동시에 닫히는 태그
        <input />
        <meta />
    ---------------------------------------------------------------
    1. 블록(block) 태그
        태그가 화면상 100% 너비를 차지하는 태그
        <h1>안녕하세요</h1>
        반갑습니다

    2. 인라인(inline) 태그
        해당 자식만큼만 너비를 차지한다
        <span></span>
        <input/>

대표적인 block 태그
    h1~h6, p , div 

대표적인 inline 태그
    span, em, strong , i , b

em, i
    두 태그 모두 자식을 기울여서 표현해준다
    em 태그는 자식이 중요하다는 정보까지 태그에 내포하고 있다
    i태그는 단순히 디자인적으로만 기울여서 표현

strong, b
    두 태그 모두 자식을 굵게 표현해준다 
    strong 태그는 자식이 중요하다는 정보까지 태그에 내포하고 있다
    b 태그는 단순히 디자인적으로만 굵게 표현

목록태그 
    목록을 나타내는 태그
    목록에는 두가지 종류가 있다 
    1. 순서가 있는 목록
        ex) 조리방법 
        ol 태그
    2. 순서가 없는 목록
        ul 태그
    
    항목을 나타내는 태그
        li 태그

ol 태그 속성
    start ( 기본값 : 1 )
        숫자 : 리스트 마커를 해당 숫자부터 시작하겠다
    type ( 기본값 : 1 )
        1 : 아라비아 숫자로 마커를 표현
        A : 대문자 순서로 마커를 표현
        a : 소문자 순서로 마커를 표현
        I : 로마자로 마커를 표현
        i : 로마자로 마커를 표현
    html5버전에서 새롭게 추가된 속성
    reversed
        속성값필요X
        순서를 거꾸로 시작

ul태그 속성
    type 
        disc : 기본값, 검정색 동그라미
        circle : 비어있는 동그라미
        square : 검정색 네모


dl 태그 (description list) 설명리스트태그

dt 태그 (description title) 설명 제목
dd 태그 (description data) 설명 데이터

    inline 태그 , 자식이 없다 
    <img />

    상업적이용 가능한 무료 이미지 
        pixabay

img 태그의 속성
    src 
        이미지 경로, 
    alt 
        이미지 설명 : 이미지에 대한 설명을 전달하는 속성
    width
        이미지 가로길이 
    height 
        이미지 세로길이
    
경로는 항상 자신이 작성하고 있는 파일이 기준이다 
. 은 자기 자신이 속한 폴더 
.. 자기 자신 폴더의 상위 폴더를 의미한다 
/ 는 안에있는 을 의미한다 
../../day01/a.jpg

자신폴더의상위, 의 상위 안에있는 day01 안에있는 a.jpg

이미지 종류
    .jpg 
        고화질의 이미지 
    .png 
        투명배경이 필요할 때
    .svg
        점, 선 , 면에 대한 정보가 코드로 저장되어있기 때문에 
        색상을 자유롭게 변경하거나 애니메이션 효과를 넣어 이미지를 
        그려줄 수 있다
