<form> 태그 안쪽에 사용할 수 있는 태그들(입력받는태그)

<input/> 태그 (비밀번호, 이메일, 날짜 등을 입력받는 태그)
    placeholder 속성
        힌트를 보여준다
<textarea> 태그 (여러줄의 글자를 입력받는 태그)
    placeholder 속성
        힌트를 보여준다
    
<select> 태그
    drop down 방식으로 옵션 중 한가지를 선택입력할 수 있도록
    만들어주는 태그들
    <option> 태그
        선택 옵션사항을 알려주는 태그
        value 속성에 값을 주면 해당 옵션이 선택되었을때
        실질적으로 입력된 값으로 인식한다
    <optgroup>태그
        여러개의 옵션 태그를 묶어주는 태그
        label 속성을 활용하여 각 그룹의 이름을 지을 수있다
